ROAD SAFTEY APP

It's an app made for swift student challenge.
This is an gamified learning app where it teaches 4 to 12 years old about road safety 
through visuals and interactive puzzel.

This is a demo app so only few levels of game is updated here.

the pros of this app is that it can be used non-english speakers also.
The app is desinged to be used without any language assitance though there is an 
option for english but it's used very minimal.
